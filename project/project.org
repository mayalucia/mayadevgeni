* Sequencing Notes

*Suggested order*:

1. *EPIC-1 & EPIC-2* (parallel) ‚Äî These are foundational for sustained
   collaboration. Without context management and project memory, long
   projects degrade.

   iterate on agents.

3. *EPIC-3* ‚Äî Agent architecture builds on evaluation capability.

4. *EPIC-5* ‚Äî Integration patterns are downstream of everything else.

5. *EPIC-6* ‚Äî Polish happens throughout but finishes last.

*Dependencies*:

- EPIC-5 benefits from all others (integration requires mature patterns)
